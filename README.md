# 🎮 Tiki Topple

A modern multiplayer adaptation of the classic **Tiki Topple** board game, rebuilt from the ground up as a real-time Android experience.

Originally developed for a hackathon, the project was later redesigned and rewritten with a stronger focus on architecture, gameplay, animations, and user experience.

## ✨ Features

* 🌐 Real-time online multiplayer
* 👤 Anonymous Authentication (Tap & Play)
* 🔄 State-based game engine
* 🎲 Complete digital implementation of the original board game mechanics
* 🎨 Custom graphics and polished UI
* ✨ Smooth animations and transitions
* 🔊 Sound effects and interactive gameplay
* ⚡ Firebase Realtime Database synchronization
* 🏆 Multi-round scoring system
* 👥 Multiplayer lobby and room management

## 🛠 Tech Stack

* Kotlin
* Android SDK
* Firebase Realtime Database
* Firebase Anonymous Authentication
* Material Design Components

## 🏗 Architecture

The project is built around a state-driven game architecture where every action updates a synchronized game state across all connected players.

Core systems include:

* Lobby Management
* Room Creation & Joining
* Multiplayer Synchronization
* Turn Management
* Secret Card Management
* Action Card System
* Score Management
* Round Management
* Game State Controller

## 📖 About the Project

Tiki Topple began as a hackathon project where the goal was to convert a physical board game into a digital application.

Although the original version was nearly complete, it fell short because of usability issues and an unfinished experience. Instead of abandoning the idea, the project was rebuilt from scratch with a cleaner architecture, better multiplayer synchronization, improved animations, anonymous authentication, and a much more polished gameplay experience.

This repository represents that rebuilt version.
